# Function-in-c

# C Function Programs

## Overview

This repository contains basic C programs demonstrating the use of functions. These programs help in understanding modular programming, function declaration, function definition, function calls, and parameter passing.

The programs are useful for:

- Beginners learning C programming
- College laboratory exercises
- Coding interview preparation
- Logic-building practice
- Understanding function concepts

---

## Programs Included

### 1. Function Without Arguments

This program demonstrates a function without parameters that displays a message.

#### Output Example

```text
Welcome to Embedded Systems
```

#### Concepts Used

- Function declaration
- Function definition
- Function call
- Void functions

---

### 2. Function With Arguments and Return Value

This program accepts two numbers, passes them to a function, and returns their sum.

#### Output Example

```text
Enter two numbers:
10
20

Sum = 30
```

#### Concepts Used

- Function parameters
- Return values
- Passing arguments
- Modular programming

---

### 3. Function With Argument Without Return Value

This program accepts a number and calculates its square using a function.

#### Output Example

```text
Enter number:
5

Square = 25
```

#### Concepts Used

- Parameter passing
- Void functions
- Function calls
